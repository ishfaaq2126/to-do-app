# To-Do List Application

## Setup

1. Clone the repository.
2. Run `docker-compose up --build`.
3. Access the application at `http://localhost:3001`.

## Alternative

1. Run cd backend - node server.js
2. Run cd frontend - npm start

## Technologies

- Backend: Node.js
- Frontend: React.js
- Database: PostgreSQL!
